1. Vector Cutter Welcome Animation

Before the idea to begin a blog

I've been trying to get the svg animation working for years. I've lost track of the number of times that I've booted Chris Coyliers classic breakdown of the technique. And every time I've hit the same hurdle, not being able to get my bitmap example image imported into a vector graphics machine and ecoded in the correct format.

This time round a combinatation of Jake Archibalds take that the Inkscape editor has a codeview mode in which you can build your image directly with code and then manipulate it with the tool (I also have Jake to thank for the boilerplate js implemenation used to get this block post up and running.) and Sara Sherman's exceptionally clear instructions for using your favourite clipart with a vinyl cutter quickly got me something on screen that looked my company logo and while not exactly optimimised looking like a solid landing page with an effecient effort effectivness co-efficient.

The main result which I had failed to grok in the past was simply that once i had exporting the full ‹path› I could simply add the fill="#13171f" and stroke="#ccc6b4" directly into the svg in my code editor and there I had functional landing page.
